\subsection{Detailed Description}
This module offers the functions to embed GuardTime timestamps in PNG image files.

{\bfseries Usage}

The timestamping functions in this module rely on the base module and any functions taking URLs as parameters additionally rely on the HTTP transport module, so the {\ttfamily libgtbase} and {\ttfamily libgthttp} libraries have to be available to your application. Refer to the usage notes in the {\ttfamily \hyperlink{gt__base_8h}{gt\_\-base.h}} and {\ttfamily \hyperlink{gt__http_8h}{gt\_\-http.h}} sections for more details.
